# CodeAlpha_Real-Time-Communication-App
"A feature-rich Real-time Video Conferencing platform built with Node.js, Socket.io, and WebRTC. Features include seamless peer-to-peer video streaming, integrated whiteboard for collaborative drawing, screen sharing capabilities, and real-time file transfer for efficient remote communication."
robust, real-time web application built for seamless remote communication, featuring high-quality video calling, interactive collaborative tools, and file sharing capabilities.
​🚀 Key Features
​Real-Time Video Calls: Peer-to-peer video streaming using WebRTC and PeerJS.
​Interactive Whiteboard: Collaborative drawing board powered by Canvas API and Socket.io.
​Screen Sharing: Built-in functionality for users to share their screens during calls.
​File Transfer: Instant file sharing between participants within the session.
​Modern UI: Sleek, glassmorphism-inspired design with a soft lavender aesthetic.
​🛠 Tech Stack
​Frontend: HTML5, CSS3 (Glassmorphism), JavaScript (EJS).
​Backend: Node.js, Express.js.
​Real-Time: PeerJS.io, WebRTC (PePeerJS)

INSTALLATION AND SETUP
1. Clone the repository:
   git clone <your-repository-link>
2. Install dependencies:
   npm install
3. Run the application:
   node server.js
4.Access the app:
 Open your browser and go to http://localhost:3000.
